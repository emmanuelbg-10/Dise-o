Elementos para incrustar contenido

Img

El elemento "img" se utiliza para mostrar imágenes en una página web. Sus atributos más habituales incluyen:

  • src: la URL de la imagen.
  • alt: texto alternativo que describe la imagen para accesibilidad y en caso de que no se cargue.
  • width y height: especifican el tamaño de la imagen.

Iframe

El elemento "iframe" se usa para incrustar contenido de otra página web dentro de un marco.

  • src: URL del contenido incrustado.
  • width y height: dimensiones del marco.
  • allow: define permisos como reproducción de audio o uso de cámara.
  • sandbox: restringe el contenido incrustado para mayor seguridad.

Embed

El elemento "embed" se utiliza para incrustar contenido externo, como archivos multimedia o aplicaciones.

  • src: la URL del recurso.
  • type: el tipo MIME del contenido.
  • width y height: dimensiones del elemento incrustado.

Object

El elemento "object" permite incrustar recursos externos, como imágenes, videos o aplicaciones interactivas.

  • data: URL del archivo a incrustar.
  • type: define el tipo MIME del contenido.
  • width y height: tamaño del objeto.

Param

El elemento "param" especifica parámetros para los elementos "object" o multimedia.

  • name: el nombre del parámetro.
  • value: el valor que se asigna al parámetro.

Video

El elemento "video" se utiliza para mostrar contenido de video.

  • src: URL del video.
  • controls: habilita controles de reproducción.
  • autoplay, loop, muted: configuran la reproducción automática, repetición o silenciado.
  • poster: imagen que se muestra antes de reproducir el video.

Audio

El elemento "audio" se emplea para reproducir contenido de audio.

  • src: URL del archivo de audio.
  • controls: muestra controles de reproducción.
  • autoplay, loop, muted: propiedades para configurar el comportamiento de reproducción.

Source

El elemento "source" proporciona diferentes fuentes para elementos multimedia como "video" o "audio".

  • src: URL de la fuente.
  • type: tipo MIME del archivo.
  • media: define condiciones de reproducción según el dispositivo.

Track

El elemento "track" añade subtítulos, descripciones o capítulos a elementos "video" o "audio".

  • src: URL del archivo de texto de subtítulos.
  • kind: tipo de texto (subtitles, captions, etc.).
  • label: título del track.
  • default: especifica el track predeterminado.

Canvas

El elemento "canvas" es un contenedor gráfico para dibujar imágenes, gráficos o animaciones mediante JavaScript.

  • width y height: tamaño del área de dibujo.
  • Suele utilizarse junto a la API de dibujo en 2D o WebGL para gráficos avanzados.

Map

El elemento "map" define un conjunto de áreas interactuables dentro de una imagen mediante un mapa de imágenes.

  • name: identifica el mapa para enlazarlo con un elemento "img".
  • Se usa junto con elementos "area".

Svg

El elemento "svg" permite crear gráficos vectoriales escalables directamente en HTML.

  • width y height: tamaño del gráfico.
  • viewBox: define el área visible.

Math

a 2 + b 2 = c 2

El elemento "math" sirve para incluir expresiones matemáticas usando el estándar MathML.

  • Suele usarse para mostrar fórmulas y símbolos matemáticos complejos.
  • Admite atributos específicos como display para la presentación en línea o en bloque.
